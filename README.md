# My Diary

## 프로젝트 소개

나의 하루를 기록하는 다이어리용 사이트입니다.
글작성 뿐만 아니라 사진추가, 기분 선택, 날씨 선택을 통해 직관적으로 하루를 기록할 수 있습니다.
반응형 웹으로 제작되었습니다.

## 사용한 기술 스택

TypeScript
